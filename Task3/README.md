AI-Driven SEO Content Cluster Generator 🚀

This repository contains a professional SEO Prompt System designed to scale content production for local businesses. It moves beyond simple article generation by using a Pillar & Cluster model to build topical authority and rank for local search terms.

Project Objective
To demonstrate a reproducible workflow that generates:

High-Authority Pillar Content: Comprehensive guides that target primary, high-volume keywords.
Semantic Content Clusters: Supporting articles that answer long-tail queries and boost internal linking.
Local SEO Optimization: Natural integration of "City + Service" keywords to capture local intent.
Conversion-Focused Copy: Content designed to turn readers into leads.

Case Study: Solar Installation (Austin, TX)
To demonstrate the system, I have generated a complete SEO Blog Pack for a residential solar provider.

1. Strategy & Intent Map
Main Topic: Solar Energy Systems
Primary Keyword: "Best Solar Panel Installation Austin" (Commercial Intent)
Supporting Keywords: "Solar tax credits Texas," "Hail damage solar panels," "Austin net metering guide."

2.  Generated Content

Pillar Blog: The Ultimate Guide to Solar in Austin

Cluster Blog A: Understanding Texas Solar Incentives

Cluster Blog B: Solar Maintenance for Central Texas Weather

Cluster Blog C: Cost Analysis of Solar vs. Utility in Austin

Key Features of the Content
H1-H3 Hierarchy: Optimized for readability and search crawlers.
Search Intent Alignment: Distinguishes between users "researching" vs. users "ready to buy."
Internal Linking Strategy: Each cluster blog links back to the Pillar to pass "Link Juice."
Local Signals: Includes mentions of local utilities (Austin Energy), climate, and regional regulations.

 How to Use This System
Open the prompts/system-prompts.md file.
Copy the Cluster Architect prompt into Gemini/ChatGPT/Claude.
Input your client's industry and target city.
Follow the sequential stages to generate a full, ready-to-publish SEO pack.
